# Vitëk Web

Браузерная версия мессенджера Vitëk. Работает как статическое SPA-приложение и подключается к тому же локальному Python-серверу, что iOS/Mac-приложение.

## Локальный запуск

1. Запустите backend:

   ```sh
   ./Server/Start\ Vitek\ Server.command
   ```

2. В отдельном окне Terminal запустите веб-версию:

   ```sh
   cd Web
   python3 -m http.server 5173
   ```

3. Откройте:

   ```text
   http://localhost:5173
   ```

Для телефона в той же Wi-Fi сети используйте адрес Mac, например:

```text
http://192.168.1.10:5173
```

В поле API укажите адрес backend-сервера, например:

```text
http://192.168.1.10:8081
```

## Что уже есть

- регистрация и вход;
- поиск пользователей по username;
- список чатов, фильтр непрочитанных;
- переписка с текстом и изображениями;
- поиск внутри активного чата;
- профиль с редактированием имени, username, почты и аватарки;
- настройки внешнего вида, приватности и поведения ввода;
- локальные пользовательские настройки и кэш состояния;
- адаптация под desktop, tablet и mobile.

Это локальная web-версия для тестирования продукта. Для публичного продакшена backend нужно переносить на HTTPS-домен и добавлять полноценные сессии, refresh tokens, rate limiting и стабильное файловое хранилище.

## Публикация

Перед публикацией поменяйте `Web/config.js`:

```js
window.VITEK_CONFIG = {
  API_BASE: "https://api.vitek.example.com"
};
```

Затем загрузите содержимое папки `Web` на статический хостинг.

Подробная инструкция: `../DEPLOY.md`.
